E-mail marketing - são arquivos HTML enviados via E-mail

-Seguem um formato padrão de construção, no qual a estrutura tem ser feita com o HTML antigo e só podemos usar as tags de tabela, imagens e links, basicamente. Várias tags não podem ser usadas como margin, position, float, background-image... entre outras, por questões de compatibilidade.

Tags de tabela: 
-tr - As linhas
-td - Os elementos 
-table - estrutura q engloba tudo

*Podemos colocar uma tag <table></table> dentro de uma tag <td></td> mas nunca uma <tr></tr> dentro da td, precisamos smp respeitar essa hierarquia! 

----ESSA HIERARQUIA-----
<table><tr>
<td><table></table></td>
</tr></table>
-------------------------

*Não esquecer de tirar o espaçamento interno padrão das tabelas(table) dessa forma, <table cellspacing="0" cellpadding="0"></table>.

*É recomendado que sempre em elementos de imagem, <img></img>, se use o display como block para q n se tenha nenhum conflito nas plataformas de E-mail, <img style="display:block;"></img> e é preciso colocar o atributo 'alt' q serve para fazer uma descrição da imagem,<img alt:"minha descrição"style="display: block;"></img>.

*Precisamos validar o documento no site da W3C validator - para confirmar q seguimos o padrão dos E-mails marketing

*OBS: Para integrar um sistema de envio de E-mail ao seu site é necessário utilizar um software e ter conhecimento sobre Back-End, as imagens para serem enviadas corretamente via E-mail precisam estar dentro de um servidor externo online.
